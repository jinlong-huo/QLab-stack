# How to Write a Paper

> 从空白页到 camera-ready 的完整流程。基于组内实际经验。

---

## Phase 0: 写之前（最重要的一步）

**先写 outline，再写段落。** 很多人打开 LaTeX 从 abstract 开始写。不要。先写 outline。

```
Outline = Section 标题 + 每个段落的 topic sentence
```

把 outline 发给导师，对齐结构之后再填内容。结构不对，文笔再好也是重写。

---

## Phase 1: 写 Figure

**Figure 是论文的骨架。** 一个高效的写法：先把所有 figure 画好，按顺序排列。然后写正文时每一段只是在解释一张图。如果一张图对应不了一段话，说明图是多余的。

---

## Phase 2: First Draft（不要停）

- 关掉拼写检查，关掉格式洁癖
- 第一遍唯一目标：把想法从脑子里倒出来
- 不要边写边改——憋住，写完一整段再回去看
- 第一遍允许写得烂。烂的第二稿 > 空白的完美稿

---

## Phase 3: Self-Edit（最痛苦但最关键）

写完放 24 小时，然后拿着 `writing-checklist.md` 逐项自查。重点：

| 检查项 | 常见问题 |
|---|---|
| Abstract | 太泛，"novel approach" — 换具体数字 |
| Intro 第 1 段 | 写了一堆背景但没说清楚"问题是什么" |
| Intro contributions | 写得太虚，"comprehensive study" — 说清楚你做了什么 |
| Method | 每一段都能对应一张图吗？如果不，缺图或废话 |
| Experiments | 有没有一个 table 是故意放进来水页数的？删掉 |
| Related Work | 是不是在罗列而不是在对比？每篇后面跟一句"Unlike [X], we..." |

---

## Phase 4: Internal Review

发给组员，用 `templates/internal-review.md` 填。**不要让别人读你还没自己改过的稿子**——浪费别人的时间。

---

## Phase 5: 最终检查

跑 `templates/writing-checklist.md`。全部勾上。投。

---

## 时间分配参考（8 周 paper）

| 阶段 | 时间 |
|---|---|
| Outline + Figures | 2 周 |
| First Draft | 2 周 |
| Self-Edit | 1 周 |
| Internal Review + Revision | 2 周 |
| Polish | 1 周 |

> 8 周出不来一个短 paper？问题通常不是写得慢，是实验还没做完就开始写了。
