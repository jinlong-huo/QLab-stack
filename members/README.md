# Members

> 个人工作区。每人一个目录，存放个人论文笔记、项目细节、实验记录。
> 这些内容**不属于组内共享资产**——离开时带走你的目录即可，不影响其他人。

## 可见性

| 目录 | 可见范围 | 说明 |
|---|---|---|
| `members/<name>/paper-notes/` | 组内 | 个人论文阅读笔记 |
| `members/<name>/projects/` | 本人 + 导师 | 项目细节、未公开 idea、实验中间结果 |
| `members/<name>/repro/` | 组内 | 实验复现记录 |

> 组内共享资产（knowledge-base / templates / onboarding / survival-guide）在顶层目录，与个人内容分离。

## 新成员

```bash
cp -r members/_template members/<your-name>
```
