# @your-name

> 个人工作区。存放论文笔记、项目细节、实验记录。

## 结构

```
members/<name>/
├── paper-notes/     ← 论文阅读笔记（每篇一文件，用 ../../paper-notes/template.md 模板）
├── projects/        ← 私人项目细节（仅本人和导师可见，用下面的模板）
└── repro/           ← 实验复现记录（用 ../../repro/template/ 模板）
```

## 离组时

运行 `../../offboarding/extract.sh <your-name>` 导出本目录。
