# Templates

组内标准模板。复制 → 填空 → 提交。不要从空白页开始。

| 模板 | 用途 | 频率 |
|---|---|---|
| `weekly-report.md` | 每周汇报 | 每周五 |
| `meeting-notes.md` | 会议记录 | 每次会议 |
| `paper-presentation.md` | 组会讲论文 | 轮到你就用 |

## 使用方式

```bash
cp templates/weekly-report.md reports/2026-W01-张三.md
# 填空
git add . && git commit -m "weekly report: 张三 W01" && git push
```
