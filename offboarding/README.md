# Offboarding

> 离组流程。目标是两件事：
> 1. **你带走你的** — 个人笔记、项目记录、实验复现，完整提取，不落东西
> 2. **组留下组的** — 共享资产不丢，你的隐性知识变成组的显性文档

## 流程概览

```
4 周前  通知导师 → 确定哪些项目交接 / 关闭 / 带着做完
2 周前  填 knowledge-handover.md → 和接手人 pair 交接
1 周前  更新 projects.md → 补 paper-notes backlog → 跑 extract.sh
最后一天  退出访谈 → 回收权限 → bye
```

## 文件

| 文件 | 用途 |
|---|---|
| [exit-checklist.md](exit-checklist.md) | 逐项打勾，不遗漏 |
| [knowledge-handover.md](knowledge-handover.md) | 隐性知识显性化模板 |
| [extract.sh](extract.sh) | 一键导出个人目录 |
