# Paper Notes

组内论文阅读笔记库。每篇论文一个 `.md` 文件，按年份分目录。

## 使用方法

1. 复制 `template.md` → `2026/论文简称.md`
2. 按模板填写 what / why / how / pros / cons
3. PR 提交

## 命名约定

`2026/第一作者姓氏-关键词.md`

示例：
- `2026/zhang-llm-inference-scheduling.md`
- `2026/kim-ocs-datacenter-topology.md`

## 为什么要写笔记

- 三个月后你不会记得这篇论文讲了什么，但笔记会
- 新人进来直接看你的笔记，比从零读论文快 10 倍
- 写 what/why/how 的过程强迫你真正理解，而不是"感觉懂了"
